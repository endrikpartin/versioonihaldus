GitHub on veebipõhine platvorm, mis võimaldab arendajatel koostööd teha tarkvaraarendusprojektidega. See pakub versioonihaldust, koodihaldust ning koostöövahendeid, mis aitavad meeskondadel ühiselt tarkvara luua, hallata ja jälgida. GitHub võimaldab kasutajatel üles laadida oma koodiprojekte (kas üksikuid faile või terviklikke projekte), hallata nende versioone, esitada muudatusi ja tagasisidet ning teha koostööd teiste arendajatega koodi arendamisel. See on populaarne platvorm avatud lähtekoodiga tarkvara projektide jaoks, kuid seda kasutavad ka ettevõtted oma privaatsete projektide haldamiseks.
GitHub Inc. asutati 2008. aastal. Selle asutajate hulka kuuluvad Tom Preston-Werner, Chris Wanstrath ja PJ Hyett. GitHubi teenus sai esmakordselt avalikkusele kättesaadavaks 2008. aasta aprillis.
